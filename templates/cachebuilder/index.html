Cache Control Panel

 Build

This action begins an incremental build of caches. It triggers automatically after any repo is updated in git. This is just a manual trigger.

 Clear

DANGER! Deletes all caches in the database and the built mods in the directory. This resets all the database environment and build dir. Rebuild cache after operation is finished will restore everything from the filesystem.

 Purge

Deletes EVERYTHING. You will need to add again all modpacks.

All the above actions are for maintainance purposes. You aren't supposed to use them normally, as Technic Antani does everything automatically. All jobs above are performed in the background and as of now no indication of progress is given.
